# Regression Equations

## Simple Regression Equation

Monthly Sales = Intercept + β1(Marketing Spend)

## Multiple Regression Equation

Monthly Sales = Intercept

* β1(Marketing Spend)
* β2(Footfall)
* β3(Inventory Availability)
* β4(Customer Rating)
* β5(Discount Percentage)
* β6(Region Dummies)
* β7(Store Type Dummies)

## Dummy Variable Explanation

Reference Categories:

* Region: East
* Store Type: Mall

The coefficients of dummy variables represent differences relative to the reference category.

## Final Model Selection

The multiple regression model was selected because it achieved the highest explanatory power and provided the strongest business insights.
